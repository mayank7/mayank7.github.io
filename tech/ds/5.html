45. What is a bubble sort and how do you perform it?

Answer:
Bubble sort is a sorting technique which can be applied to data structures like arrays. Here, the adjacent values are compared and their positions are exchanged if they are out of order. The smaller value bubbles up to the top of the list, while the larger value sinks to the bottom.

49. What is a queue ?

Answer:
A Queue refers to a sequential organization of data. It is a FIFO type data structure in which an element is always inserted at the last position and any element is always removed from the first position.
