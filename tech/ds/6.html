51. What is a postfix expression?

Answer:
It is an expression in which each operator follows its operands. Here, there is no need to group subexpressions in parentheses or to consider operator precedence..

54. Define in brief an array. What are the types of array operations?

Answer:
An array is a set of homogeneous elements. Every element is referred by an index. Arrays are used for storing the data until the application expires in the main memory of the computer system. So that, the elements can be accessed at any time. The operations are: Traversal, Deletion, Insertion.
