(Q7.) Write a script to generate the multiplication table of the number entered by the user.

Answer:
$vi table.sh
echo "Enter the number : "
read n
for((i=1; i<11; i++))
do
echo $n " * " $i " = " $(($n*$i))
done

For Output : $sh table.sh
