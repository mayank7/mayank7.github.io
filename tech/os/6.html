(Q6.) Write a script to generate the following pattern where maximum limit is entered by the user.

Answer:
$vi pattern.sh
echo "Enter the maximum limit : "
read n
for((i=1; i<$n+2; i++))
do
for((j=1; j<i; j++))
do
echo -n $j
done
echo " "
done

For Output : $sh pattern.sh

Enter the maximum limit : 4
